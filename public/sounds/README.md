# Audio Files untuk Timer

Folder ini berisi file audio yang digunakan oleh sistem timer.

## File yang Diperlukan:

1. **warning.mp3** - Suara "Tet!" (pendek) - Diputar saat sisa 60 detik
2. **countdown.mp3** - Suara "Bip" (pendek) - Diputar saat sisa 5, 4, 3, 2, 1 detik
3. **finish.mp3** - Suara "Biiiiiip" (panjang/keras) - Diputar saat transisi CLIMBING → REST
4. **start.mp3** - Suara peluit/buzzer - Diputar saat transisi REST → CLIMBING

## Cara Mendapatkan File Audio:

### Opsi 1: Gunakan Online Sound Generator
- Kunjungi https://www.onlinetonegenerator.com/
- Generate tone sesuai kebutuhan
- Download sebagai MP3

### Opsi 2: Gunakan Free Sound Library
- Kunjungi https://freesound.org/
- Cari: "beep", "buzzer", "alarm", "notification"
- Download file dalam format MP3 atau WAV (konversi ke MP3 jika perlu)

### Opsi 3: Rekaman Manual
- Gunakan aplikasi perekam suara
- Rekam suara sesuai kebutuhan
- Export sebagai MP3

## Rekomendasi Format:
- Format: MP3 (preferred) atau WAV
- Durasi: 0.5-2 detik (kecuali finish.mp3 bisa lebih panjang)
- Bitrate: 128 kbps atau lebih tinggi
- Sample Rate: 44.1 kHz

## Catatan:
Jika file audio tidak ada, sistem akan tetap berfungsi tetapi tanpa suara. Tidak akan ada error, hanya silent.
